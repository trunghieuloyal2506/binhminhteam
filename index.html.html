🔥 Kết nối Firebase

Sau khi kết nối, mọi thay đổi sẽ được đồng bộ real-time cho toàn team. Chỉ cần làm 1 lần.

1
Vào console.firebase.google.com → Tạo project mới → đặt tên bất kỳ (VD: canban-team)
2
Vào Realtime Database → Create database → chọn Asia (Singapore) → chọn Test mode → Enable
3
Copy Database URL (dạng https://xxx.asia-southeast1.firebasedatabase.app) dán vào bên dưới
4
Vào Project Settings ⚙️ → tab General → kéo xuống Your apps → nhấn </> Web → đăng ký app → copy apiKey và projectId
